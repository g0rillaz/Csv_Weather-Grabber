# Csv_Weather-Grabber
added the filename as a new Colum to the Dataframe.


Get started

-- Setup the Enviroment
required:
          - python3.0+
          - virtualenv
          
          
          
When "virtualenv" is not installed you need to install.

On Windows:
# python -m pip install --user virtualenv
On Linux:
# python3 -m pip install --user virtualenv




--Create a enviroment an start it

Create an Enviroment
# python3 -m venv py3env

-- Start your enviroment
On Windows
# py3env\Scripts\activate.bat
On Linux
# source py3env/bin/activate

Install Pandas in your Enviroment
# pip install pandas

Start the script
example:
# python C:\CsvWeathercompress.py
